Core Features

Everything you need.
Nothing you don't.

Four precisely built modules that cover the full navigation workflow — from reading to research to rapid pagination.

Jump to Top & Bottom

One click to the start or end of any page. Buttons intelligently disable themselves when you're already there — so you always have visual confirmation of your position.

Detects 6 URL patterns automatically — no configuration needed. If URL patterns don't match, it falls back to DOM-based rel="next" detection.

Type 3+ characters and every match highlights in amber across the whole page. Navigate with Enter / Shift+Enter. Press Escape to clear.

Pagination Engine · Phase 1

Six URL patterns.
Detected instantly.

The engine scans the current URL and automatically constructs the previous and next page URL — no clicking, no scrolling for buttons.

Pattern · Query Param
thehindu.com?page=11
→ next page
thehindu.com?page=12
Used by: The Hindu, news sites
Pattern · Short Query
udemy.com?p=5
→ next page
udemy.com?p=6
Used by: Udemy, e-commerce
Pattern · WordPress Style
blog.com/page/11/
→ next page
blog.com/page/12/
Used by: WordPress blogs, dev sites
Pattern · WordPress paged
site.com?paged=3
→ next page
site.com?paged=4
Used by: WordPress archives
Pattern · Forum Style
forum.com?pg=7
→ next page
forum.com?pg=8
Used by: Forums, communities
Phase 2 · DOM Fallback
No URL pattern found
→ scans DOM for
rel="next" & rel="prev"
Fallback for: Custom pagination

Roadmap

This is just the start.

Version 1 ships the core engine. Versions 2 and 3 add AI-powered detection and site-wide multi-page search.

v1.0 — Current
Core Navigation Engine
  • ✓ 6 URL pagination patterns
  • ✓ DOM rel=next/prev fallback
  • ✓ Full-page text search + highlights
  • ✓ Enter/Shift+Enter match navigation
  • ✓ Draggable, position-persistent widget
  • ✓ Ctrl+Shift+N keyboard toggle
  • ✓ Search history saved locally
  • ✓ Shadow DOM — zero style conflicts
v2.0 — Upcoming
Advanced Search & Controls
  • ○ Case-sensitive search toggle
  • ○ Whole-word matching
  • ○ Regex search mode
  • ○ Infinite scroll detection
  • ○ Keyboard-only navigation (J/K)
  • ○ Auto-detection of "Load More" buttons
v3.0 — Vision
AI-Powered Site Search
  • ○ AI pagination: detect "Older Posts", "Next →", "Load More"
  • ○ Multi-page search (search across 10 pages)
  • ○ Results aggregated by page number
  • ○ Site-wide domain crawl mode
